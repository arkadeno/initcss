initcss

Smart defaults for styling websites

initialize |iˈniSHəˌlīz| – set to the value or put in the condition appropriate to the start of an operation: the counter is initialized to one.

Use these base styles for body, typography, buttons, forms, and tables in combination with Normalize CSS and other libraries like Basscss or Tachyons to quickly build out sites without the bloat of larger CSS frameworks.
